기밀 문서 (블러 시연용)

이 영역은 실제 확장에서 보호되는 화면을 흉내냅니다. 위 판정이 위험(60점 이상)이 되는 순간 즉시 블러 처리되고, 위협이 사라지면 약 2초 뒤 자동 해제됩니다.

🛡 제3자 응시 감지 — 화면 보호 중

크롬 확장으로 실제 사용하기

  1. 확장 패키지 다운로드 → 압축 해제
  2. chrome://extensions → 우상단 개발자 모드 ON
  3. 압축해제된 확장 프로그램 로드 → 폴더 선택
  4. 설치 시 열리는 페이지에서 카메라 권한 허용 → 툴바 아이콘에서 토글 ON
