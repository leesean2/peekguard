# PeekGuard 데스크톱 — 브라우저 밖 전체 화면 보호

크롬 확장은 브라우저 탭만 가릴 수 있다. 이 Electron 트레이 앱은 같은 감지
엔진으로 **OS 화면 전체**(한글·엑셀·메신저 등 모든 앱)를 보호한다.

## 실행

```bash
cd desktop
npm install
npm start
```

트레이(작업 표시줄 우측)에 PeekGuard 아이콘이 생기고 즉시 감시를 시작한다.
Windows 가 카메라를 막으면: 설정 > 개인정보 > 카메라 > **데스크톱 앱 허용** ON.

## 동작

- 위협(danger, 60점+) 감지 → **모든 모니터**에 최상위 오버레이 표시
  - **블러 모드**: Windows 11 아크릴로 실제 화면을 블러 + 판정 근거(신호·점수) 표시
  - **디코이 모드**: 평범한 가짜 문서 화면 (보안 UI 없음)
- 위협 소실 → 약 2초(해제 히스테리시스) 뒤 자동 해제
- 탈출구: **Esc 2연타**(1초 내) 또는 오버레이의 버튼 → 5분 일시정지
- 트레이 메뉴: 활성화 토글 · 민감도(둔감/보통/민감) · 가림 방식 · 로그인 시 자동 시작

## 구조 (확장과 1:1 대응)

| 데스크톱 | 확장 | 역할 |
|---|---|---|
| `main.js` | `background.js` | 상태 소유 + 중계, 오버레이 수명주기 |
| `capture.html/js` (숨김 창) | `offscreen.html/js` | 웹캠 → pico → 엔진 |
| `overlay.html` (전체 화면 창) | `content.js` | 블러/디코이 표시 |
| `lib/engine.js` | `engine.js` | **동기화 사본** — 판정 로직 원본은 extension/ |

`lib/engine.js`, `lib/pico.js`, `models/facefinder` 는 `extension/` 의 동기화
사본이다. 엔진을 수정하면 세 위치(extension, web/lib, desktop/lib)에 모두 반영할 것.

## 검증

```bash
npm run smoke   # 카메라 → 감지 → 첫 리포트 수신까지 확인 후 자동 종료
```

## 정직한 한계

- 전면 카메라(웹캠) 화각 밖은 감지 불가 — 확장·웹과 동일.
- 오버레이는 화면을 가리지만, 위협자가 이미 본 내용을 되돌릴 수는 없다.
- Windows 10 이하에서는 아크릴 블러 대신 불투명 가림으로 동작한다.
